Linux manual pages Section 3  

Name

PCRE — Perl-compatible regular expressions

Synopsis

#include <pcre.h>
void pcre_free_study( pcre_extra *extra);
 
void pcre16_free_study( pcre16_extra *extra);
 

DESCRIPTION

This function is used to free the memory used for the data generated by a call to pcre[16]_study() when it is no longer needed. The argument must be the result of such a call.

There is a complete description of the PCRE native API in the pcreapi(3) page and a description of the POSIX API in the pcreposix(3) page.
