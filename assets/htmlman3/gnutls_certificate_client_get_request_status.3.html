Linux manual pages Section 3  

Name

gnutls_certificate_client_get_request_status — API function

Synopsis

#include <gnutls/gnutls.h>
int gnutls_certificate_client_get_request_status( gnutls_session_t session);
 

ARGUMENTS

gnutls_session_t session

is a gnutls session

DESCRIPTION

Get whether client certificate is requested or not.

RETURNS

0 if the peer (server) did not request client authentication or 1 otherwise, or a negative error code in case of error.
